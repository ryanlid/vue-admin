# vue-admin-template
## 初始化项目

### @vue/cli 初始化项目

```
vue create vue-admin
```

>```bash
>features: Babel, Router, Vuex, Linter
>mode for router: hash
>linter config: Prettier
>lint features: Lint on save
>placing config for Babel, PostCSS, ESLint, etc.? In package.json
>```

### 添加第三方依赖包

```
yarn add axios
yarn add element-ui
```
